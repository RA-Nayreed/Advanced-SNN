# Brain Viewer

Static Three.js viewer for Advanced-SNN snapshot NDJSON files.

Run a local static server from this directory:

```bash
python3 -m http.server 5173
```

Open `http://127.0.0.1:5173`, then load a snapshot generated by the Rust CLI:

```bash
cargo run --release -- brain --neurons 2000 --fanout 32 --steps 200 --seed 7 \
  --snapshot-out brain_blob.ndjson --snapshot-every 2 --snapshot-neurons 1500 --snapshot-synapses 4000
```
